# Rana Nisar - Portfolio Website

A professional and responsive portfolio website showcasing skills, projects, and contact information.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Easy Navigation**: Sticky navigation bar with smooth scrolling to sections
- **Sections**:
  - Home: Hero section with introduction
  - About: Personal background and expertise
  - Skills: Technical skills organized by category
  - Projects: Showcase of work and projects
  - Contact: Contact information and links

## Live Demo

Visit the website at: https://rananisarsb51214-web.github.io/Rananisar-portifolio-website-/

- HTML5
- CSS3
- Responsive Design
